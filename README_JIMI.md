# Jimi Image

Place your `jimiround.png` file (a round picture of your dog Jimi) in this directory.

The image should be:
- Circular/round format
- Recommended size: 120x120px or larger
- PNG format with transparency if possible

The image will appear at the bottom of the tutorial panel and link to your GitHub project.

To update the GitHub link, edit the href in index.html:
```html
<a href="https://github.com/yourusername/fuk-point_worldtext" target="_blank" class="jimi-link">
```

Replace `yourusername` with your actual GitHub username.
